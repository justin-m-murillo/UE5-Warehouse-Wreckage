*UE5 training project*

This is a training project from Udemy.

1. Blueprint basics:

  * pins
2. Programming basics in Blueprint:

  * strings  
  * references  
  * functions
3. Unreal basics:  
  * maps  
  * actors  
  * components  
  * transforms  
  * vectors
4. OOP basics in Unreal:  
  * objects/structs  
  * classes  
  * member functions
